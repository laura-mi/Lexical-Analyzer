Introduce into the Input.txt file the input you need to parse.

Run the analyzer (Double click on Analyzer.bat)

In the tests folders you can find a few examples of inputs.

Dependencies:
1. Install flex: https://sourceforge.net/projects/winflexbison/files/
2. Install gcc compiler. You can find it around here: http://www.mingw.org/
3. Add the path for win_flex in your environment variables (just if you intend to update the rules)
